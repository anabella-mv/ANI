            Se nos presenta la siguiente sumatoria: 
            \begin{equation*}
                \sum_{k=1}^{n} \frac{k}{\left(k+1\right)!}
            \end{equation*}
            como se nos pide el desarrollo de una fórmula es posible desglozar la sumatoria para identificar un patrón o ver cada termino individualmente para poder trabajar en otra forma de escribirlo:
            \begin{equation*}
                \frac{1}{2!} + \frac{2}{3!} + \frac{3}{4!} + \cdots + \frac{n}{\left(n+1\right)!}
            \end{equation*} 
            Creo es más cómodo trabajar con cada término de la sumatoria, en este caso, puedo ver que los términos los puedo reescribir usando una diferencia para simplificar algunos términos y concatenadamente eliminar otros:
            \begin{equation*}
                \frac{k}{\left(k+1\right)!} = \frac{\left(k+1\right) - 1}{\left(k+1\right)!} = \frac{k+1}{\left(k+1\right)!} - \frac{1}{\left(k+1\right)!} = \frac{1}{k!} - \frac{1}{\left(k+1\right)!}
            \end{equation*}
            Con esto, es posible reescribir la sumatoria como:
            \begin{equation*}
                \sum_{k=1}^{n} \left( \frac{1}{k!} - \frac{1}{\left(k+1\right)!} \right) = \left( \frac{1}{1!} - \frac{1}{2!} \right) + \left( \frac{1}{2!} - \frac{1}{3!} \right) + \cdots + \left( \frac{1}{n!} - \frac{1}{\left(n+1\right)!} \right)
            \end{equation*}
            Con esto la sumatoria es telescópica (terminos que se cancelan), para esto es necesario escribir cada término de la sumatoria como una diferencia de fracciones. Por lo tanto, la fórmula para la sumatoria es:
            \begin{equation*}
                \sum_{k=1}^{n} \left( \frac{1}{k!} - \frac{1}{\left(k+1\right)!} \right) = \frac{1}{1!} - \frac{1}{\left(n+1\right)!}
            \end{equation*}
             Ahora, es posible verificar la fórmula obtenida, para esto es necesario evaluar la sumatoria. Para un valor específico de $n$, por ejemplo, para $n=3$:
            \begin{equation*}
                \sum_{k=1}^{3} \frac{k}{\left(k+1\right)!} = \frac{1}{2!} + \frac{2}{3!} + \frac{3}{4!} = \frac{1\cdot 3 \cdot 4 + 2\cdot 4 + 3}{4!}= \frac{12+8+3}{24}= \frac{23}{24}
            \end{equation*}
            Comparando esto con la fórmula obtenida:
            \begin{equation*}
                \frac{1}{1!} - \frac{1}{\left(3+1\right)!} = 1 - \frac{1}{4!} = 1 - \frac{1}{24} = \frac{23}{24}
            \end{equation*}
            Con esto hay mas seguridad de que la fórmula obtenida es correcta, además, es posible evaluar para otros valores de $n$ para verificar la fórmula obtenida.

    \item Determinar la longitud de la diagonal de un ortoedro conociendo las longitudes de sus aristas.\\\\
    Los ortoedros son figuras tridimensionales con seis caras rectangulares, y sus aristas se pueden denotar como $a$, $b$ y $c$ (ya que coinciden de a pares en cada cara del ortoedro). La diagonal de un ortoedro se puede calcular utilizando el teorema de Pitágoras en tres dimensiones teniendo en cuenta que toma una de las aristas (sea esta $a$), la diagonal de una cara del ortoedro (como es de una cara donde no está $a$, resulta esa diagonal del triángulo formado por las aristas $b$ y $c$ $\implies d_1=\sqrt{b^2 +c^2}$) y resulta la hipotenusa de ese triangulo rectángulo la diagonal del ortoedro. La fórmula para la longitud de la diagonal $d$ es:
        \begin{equation*}
            d= \sqrt{a^2 + \left(\sqrt{b^2+c^2}\right)^2} = \sqrt{a^2 + b^2 + c^2}
        \end{equation*}
    Dependiendo que arista se tome como base, el resultado es el mismo, por lo tanto, la fórmula para la diagonal de un ortoedro con aristas $a$, $b$ y $c$ es la misma.

    \item Dado un prisma recto de base cuadrada, encontrar la longitud del lado de la base y la altura conociendo su volumen, $63cm^3$, y su área total $102cm^2$\\\\
    Para resolver este problema, primero denotamos la longitud del lado de la base cuadrada como $s$ y la altura del prisma como $h$. El volumen de un prisma recto se calcula como el área de la base multiplicada por la altura, es decir, $V = A_{base} \cdot h$. Dado que la base es un cuadrado, su área se calcula como $A_{base} = s^2$. Por lo tanto, el volumen se puede expresar como:
    \begin{equation*}
        V = s^2 \cdot h
    \end{equation*}
    Dado que el volumen es de $63cm^3$, podemos escribir la ecuación:
    \begin{equation*}
        s^2 \cdot h = 63
    \end{equation*}
    Por otro lado, el área total de un prisma recto se calcula como la suma del área de las dos bases y el área lateral. El área de las dos bases es $2 \cdot A_{base} = 2s^2$, y el área lateral se calcula como el perímetro de la base multiplicado por la altura, es decir, $A_{lateral} = P_{base} \cdot h$. Dado que la base es un cuadrado, su perímetro se calcula como $P_{base} = 4s$. Por lo tanto, el área lateral se puede expresar como:
    \begin{equation*}
        A_{lateral} = 4s \cdot h
    \end{equation*}
    Por lo tanto, el área total del prisma se puede expresar como:
    \begin{equation*}
        A_{total} = 2s^2 + 4s \cdot h = 102
    \end{equation*}
    Ahora tenemos un sistema de dos ecuaciones con dos incógnitas:
    \begin{gather*}
        s^2 \cdot h = 63 \quad (1)\\
        2s^2 + 4s \cdot h = 102 \quad (2)
    \end{gather*}
    Podemos resolver este sistema de ecuaciones para encontrar los valores de $s$ y $h$. Primero, despejamos $h$ de la ecuación (1):
    \begin{equation*}
        h = \frac{63}{s^2}
    \end{equation*}
    Luego, sustituimos este valor de $h$ en la ecuación (2):
    \begin{equation*}
        2s^2 + 4s \cdot \frac{63}{s^2} = 102
    \end{equation*}
    Simplificando esta ecuación, obtenemos:
    \begin{equation*}
        2s^2 + \frac{252}{s} = 102
    \end{equation*}
    Multiplicando ambos lados de la ecuación por $s$ para eliminar el denominador
    \begin{equation*}
        2s^3 + 252 = 102s
    \end{equation*}
    Reorganizando esta ecuación, obtenemos:
    \begin{equation*}
        2s^3 - 102s + 252 = 0
    \end{equation*}
    Dividiendo toda la ecuación por 2 para simplificarla, obtenemos:
    \begin{equation*}
        s^3 - 51s + 126 = 0
    \end{equation*}
    Ahora, podemos resolver esta ecuación cúbica para encontrar el valor de $s$. Podemos intentar encontrar una raíz racional utilizando el teorema de las raíces racionales o utilizando métodos numéricos. Probando con algunos divisores de 126, encontramos que $s=3$ es una raíz de la ecuación (desglozando el 51s en 9s+42s, es facil darse cuenta de donde viene (agregar $3s^2-3s^2$)). Por lo tanto, podemos factorizar la ecuación como:
    \begin{equation*}
        (s - 3)(s^2 + 3s - 42) = 0
    \end{equation*}
    Ahora, podemos resolver cada factor para encontrar las posibles soluciones para $s$. De la primera factor, obtenemos $s=3$. De la segunda factor, podemos usar la fórmula cuadrática para encontrar las raíces:
    \begin{equation*}
        s = \frac{-3 \pm \sqrt{3^2 - 4 \cdot 1 \cdot (-42)}}{2 \cdot 1} = \frac{-3 \pm \sqrt{9 + 168}}{2} = \frac{-3 \pm \sqrt{177}}{2}
    \end{equation*}
    Dado que $s$ representa la longitud del lado de la base, debe ser un número positivo. Por lo tanto, descartamos la raíz negativa y nos quedamos con:
    \begin{equation*}
        s = \frac{-3 + \sqrt{177}}{2}
    \end{equation*}
    Ahora que tenemos el valor de $s$, podemos sustituirlo en la ecuación (1) para encontrar el valor de $h$:
    \begin{equation*}
        h = \frac{63}{s^2} = \frac{63}{\left(\frac{-3 + \sqrt{177}}{2}\right)^2}
    \end{equation*}
    Por lo tanto, la longitud del lado de la base es $s = \frac{-3 + \sqrt{177}}{2}$ cm y la altura del prisma es $h = \frac{63}{\left(\frac{-3 + \sqrt{177}}{2}\right)^2}$ cm.
